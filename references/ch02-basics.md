# JLREQ 第2章 日本語組版の基礎

> 出典: W3C JLREQ (Requirements for Japanese Text Layout) §2 Basics
> URL: https://www.w3.org/TR/jlreq/#basics

---

## 2.1 日本語組版に用いる文字

### 文字種
- **漢字**: 意味を表す表意文字。常用漢字約2,000字程度を標準的に使用
- **仮名**: ひらがな・カタカナの2種。音を表す表音文字
  - ひらがな: 日本語の文法的要素（助詞・助動詞・語尾等）に使用
  - カタカナ: 外来語・擬音語・強調等に使用
- **欧字**: ラテンアルファベット。欧文（横組み）として混在配置
- **約物**: 句読点・括弧類・ダッシュ類等

### 文字サイズ体系
- 日本語組版では文字サイズを「級」(Q) または「ポイント」(pt) で指定
- 1級 = 0.25mm、1pt ≈ 0.3528mm
- 基準文字サイズ（基本版面の文字サイズ）を「級数」で呼ぶ
- 級数とポイントの近似: 1Q ≈ 0.71pt（例: 13Q ≈ 9pt、15Q ≈ 11pt）

### 字面と字胴
- **字面（じめん）**: 文字の可視部分。字胴内の中心に配置
- **字胴（じどう）**: 文字を囲む仮想的なボックス。正方形を基本とする
- CSS: `font-size` が字胴サイズに対応

## 2.2 基本版面（きほんはんめん）

### 基本版面の構成要素
- **版面（はんめん）**: 文字や図版が配置される領域
- **丁（ちょう）**: 用紙1枚（片面または両面）
- **柱（はしら）**: 版面の上下左右に配置する余白部分
  - 柱にはノンブル（ページ番号）や柱（running head）を配置

### 版面の設計手順
1. 用紙サイズの決定
2. 基本版面のサイズ（行字数 × 行数）の決定
3. 版面の配置位置の決定（天・地・のど・小口の余白設定）

### CSS対応プロパティ
- 用紙サイズ: `@page { size: <width> <height>; }`
- 版面余白: `@page { margin: <top> <right> <bottom> <left>; }`
- 行字数: `line-length`（概念的。実装ではフォントサイズと幅の組み合わせ）
- 行数: `line-count`（概念的）

## 2.3 縦組みと横組み

### 縦組み（たてぐみ）
- 行の進行方向: 上→下
- 行の積み方向: 右→左
- CSS: `writing-mode: vertical-rl;`
- 主に小説・新聞・学術論文等で使用
- 欧字・数字は時計回りに90度回転配置（`text-orientation: mixed;`）

### 横組み（よこぐみ）
- 行の進行方向: 左→右
- 行の積み方向: 上→下
- CSS: `writing-mode: horizontal-tb;`（デフォルト）
- 主にビジネス文書・技術文書・Webページ等で使用

### 縦組みと横組みの主な違い

| 項目 | 縦組み | 横組み |
|------|--------|--------|
| 行の進行方向 | 上→下 | 左→右 |
| 行の送り方向 | 右→左 | 上→下 |
| 開き方向 | 右開き | 上開き（または左開き） |
| 句読点 | 、。（行中央） | 、。（左下寄り） |
| 括弧の向き | 通常向き | 回転 |
| 数字の配置 | 横倒し or 縦中横 | 通常配置 |
| 強調（圏点） | 側に配置 | 側に配置（下側） |
| ルビ | 左側 | 上側 |

### CSS制御
```css
/* 縦組み */
writing-mode: vertical-rl;
text-orientation: mixed;  /* 欧字を回転 */

/* 横組み */
writing-mode: horizontal-tb;
```

## 2.4 基本版面の指定

### 版面サイズの設計
1. **行字数の決定**: 1行に入る文字数（例: 40字詰め）
2. **行数の決定**: 1ページに入る行数（例: 45行）
3. **行間の設定**: 行のびき（行間）を設定
   - 行間 = 行の高さ − 文字サイズ
   - 一般的に行間は文字サイズの50%〜70%

### 余白の原則
- **天（てん）**: 版面上端の余白
- **地（ち）**: 版面下端の余白
- **のど**: 版面綴じ側の余白
- **小口（こぐち）**: 版面開き側の余白

#### 余白比率の目安
- 天 < 地（天を狭く、地を広く）
- のど < 小口（のどを狭く、小口を広く）
- 一般的: 地 = 天 × 1.5〜2倍、小口 = のど × 1.5〜2倍

### 基本版面の配置
- 縦組み: 版面を用紙中央よりやや右上に配置
- 横組み: 版面を用紙中央よりやや上に配置

### CSS対応
```css
@page {
  size: A5;  /* 用紙サイズ */
  margin-top: 20mm;    /* 天 */
  margin-bottom: 30mm; /* 地 */
  margin-inside: 18mm; /* のど */
  margin-outside: 22mm; /* 小口 */
}
```

## 2.5 柱（ヘッダー）とノンブル（ページ番号）

### 柱（running head）の原則
- 柱は版面の外側（天または小口の余白部分）に配置
- 縦組み: 版面の左右（小口側）に配置
- 横組み: 版面の上部に配置

### 柱に含める内容
- 書名・章名・節名・ページ番号等
- 見開き左右で内容を交互に配置（偶数頁・奇数頁で異なる内容）

### ノンブル（ページ番号）の原則
- ノンブルは柱の一部として配置
- 縦組み: 通常、小口側の天または地に配置
- 横組み: 通常、ページ上部または下部の端に配置

### 偶数頁・奇数頁の配置
- **偶数頁（左ページ）**: ノンブルは左端、柱は左側に配置
- **奇数頁（右ページ）**: ノンブルは右端、柱は右側に配置
- 見開きで対称的な配置を基本とする

### 柱の文字サイズ
- 本文より1〜2ポイント小さい文字サイズを使用
- 一般的に本文の70%〜80%程度

### CSS対応
```css
@page :left {
  /* 偶数頁（左ページ） */
  @top-left { content: counter(page); }
  @top-right { content: "書名"; }
}

@page :right {
  /* 奇数頁（右ページ） */
  @top-left { content: "章名"; }
  @top-right { content: counter(page); }
}
```

## 要点まとめ

1. **文字体系**: 漢字・ひらがな・カタカナの3種を適切に使い分ける
2. **基本版面**: 行字数・行数・余白の3要素で設計。天<地、のど<小口の原則
3. **縦組み/横組み**: `writing-mode` で制御。句読点・括弧・数字の扱いが異なる
4. **版面設計**: 用紙→基本版面→余白の順で設計。余白比率は定石に従う
5. **柱・ノンブル**: 見開きで対称配置。偶数頁/奇数頁で内容を交互に配置
