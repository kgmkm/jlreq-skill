# JLREQ 第3章 行の組版処理 — AIエージェント向けリファレンス

> 出典: W3C JLREQ (https://www.w3.org/TR/jlreq/#line_composition)
> CSS仕様への対応: CSS Writing Modes L3/L4, CSS Text L3/L4, CSS Inline Layout L3, CSS Ruby Layout L1

---

## 3.1 約物などの組版処理

### 3.1.1 縦組と横組で異なる約物

- 縦組で回転する約物: 欧文読点 `,` 欧文ピリオド `.` 疑問符 `?` 感嘆符 `!` コロン `:` セミコロン `;`
  - 縦組では時計回りに90度回転して配置する
- 縦組と横組で形が異なる約物: 始め二重括弧 `〝` 終わり二重括弧 `〟`
- 終わりダッシュ `―` とリーダー `…` は横組では字面の左端、縦組では字面の上端に詰めて配置する
- CSS: `text-orientation: mixed` がデフォルトで約物の自動回転を制御

### 3.1.2 句読点・括弧類の基本配置

| 種類 | 横組での位置 | 縦組での位置 |
|------|-------------|-------------|
| 句点 `。` | 字面の左上 | 字面の右上 |
| 読点 `、` | 字面の左上 | 字面の右上 |
| 始め括弧 `(「【` など | 字面の右半分 | 字面の下半分 |
| 終わり括弧 `)」】` など | 字面の左半分 | 字面の上半分 |

- 句読点の字幅: 組版方向に対して字幅の1/2を占める（半角幅）。残りの1/2は行間側に空く
- CSS: `font-feature-settings: "halt"` で句読点を半角幅に詰められる（OpenType対応フォント）

### 3.1.3 読点・中点の例外配置

- 行頭に読点・中点が来た場合: 前行末に配置する（前行末禁則に優先）
  - 行末に約物が連続する場合はさらに前へ送る
- 読点・中点の字幅: 行頭・行末では半角幅、行中では全角幅で配置する場合がある
- 行末で幅1/2の読点・中点が来た場合: 前の文字に詰めて配置

### 3.1.4 始め括弧・終わり括弧・読点・句点・中点の連続

連続する約物の配置ルール:
- 始め括弧同士の連続: 各々の約物の字幅を1/2にして詰める
- 終わり括弧+句読点: 字幅1/2ずつで合計1文字分に収める
- 終わり括弧同士の連続: 字幅を詰める（最多3連続まで詰める例あり）
- 句読点同士の連続: 通常は発生しないが、文末と引用内文末が接続する場合は字幅を詰める
- 優先順位: 始め括弧 > 終わり括弧 > 句読点 > 中点（詰めの優先度）

### 3.1.5 行頭の始め括弧類の配置

- 行頭に来た場合、字下げを省略して行頭に配置する
- ただし次の場合は字下げを省略しない:
  - 段落先頭の始め括弧（インデントが必須の文脈では字下げする）
  - 見出しなど特別な組版スタイルの場合
- CSS: `text-spacing-trim: space-first`（デフォルト）が行頭字下げ省略を制御
- CSS Text L4: `text-autospace` による自動スペース調整も関連

### 3.1.6 区切り約物・ハイフン

- 区切り約物: ダッシュ類 `—–`、三点リーダー `…`、全角ハイフン `-`
- 行頭に配置可能（行頭禁則の対象外）
- 行末で分割可能（ダッシュは前行末に残し、継続を示す）
- ハイフン類:
  - 全角ハイフン `-`: 和文扱い、行頭禁則
  - 半角ハイフン `-`: 欧文扱い、行頭・行末の制約は緩い
  - 二重ハイフン `＝`: 和文扱い

### 3.1.7 行頭禁則（ぶら下げ禁則）

行頭に配置してはならない文字:
- 終わり括弧類: `)」】〉》〕〗〙〛』›）：閉じ括弧全般
- ピリオド `.` カンマ `,` 欧文疑問符 `?` 感嘆符 `!`
- コロン `:` セミコロン `;`
- 和文の句読点: `。，、・：；`
- 中点 `・`
- 終わりダッシュ `―`（前行末にぶら下げる）
- リーダー `…`（前行末にぶら下げる）

### 3.1.8 行末禁則

行末に配置してはならない文字:
- 始め括弧類: `（「【〈《〔〖〘〚『‹：開き括弧全般
- 前置省略記号: `￥＄＃￡`
- ルビの開始位置としての親文字

### 3.1.9 行末の終わり括弧・句読点・中点

- 行末に収まらない場合: 次の行頭に配置し、前行末は空きを生じさせる
- ただし字幅1/2の約物が連続する場合: 行末に収まるまで詰めて配置可能
- 行末で終わり括弧+句読点が来た場合: 両方とも前行末に収めることが可能

### 3.1.10 分割禁止（分離してはならない文字のペア）

連続して配置されるべき文字（分割禁止ペア）:
- 欧文間隔を除く和文文字同士は原則として分割しない
- 具体的な分割禁止条件:
  - 始め括弧 + 直後の文字: 括弧と中身を分離しない
  - ルビ付き親文字群: 親文字とルビを分離しない
  - 割注の開始・終了位置
  - 組文字: 〳〴〵など

### 3.1.11 行調整で字間を空けない箇所

行調整時に字間を空けてはならない箇所:
- 和文文字と欧文文字の間（欧文スペースを除く）
- 和文文字とアラビア数字の間
- 欧文文字とアラビア数字の間
- アラビア数字同士の間
- 単語内の欧文文字間
- 和文文字と約物の間（約物が字幅1/2の場合）
- 割注内の文字間

### 3.1.12 行調整処理例

行調整の優先順位（詰め・空けの判断順）:
1. 字間を空けてはならない箇所は空けない
2. 約物の字幅を詰める（半角→さらに詰める）
3. 和文文字間の空きを調整
4. 欧文スペースの幅を調整（最後の手段）

## 3.2 和欧文混植処理・縦中横

### 3.2.1 和文と欧文の混植

- 和文と欧文を混植する場合、原則として和欧文間隔（1/4角相当）を挿入する
- CSS: `text-autospace: auto`（デフォルト）で自動挿入
  - `text-autospace: no-autospace` で無効化可能
- 和文と欧文の間に欧文スペースがある場合は和欧文間隔を省略する
- 和文とアラビア数字の間も同様に和欧文間隔を挿入

### 3.2.2 横組の和欧文混植文字

- 欧文はベースラインで和文と揃える
- 和文と欧文の文字サイズを揃える場合、欧文のエムサイズは和文の字面サイズに合わせる
- 欧文のベースライン位置: 和文の仮想ボディの下端に近い位置

### 3.2.3 縦組の和欧文混植文字

- 欧文を縦組に配置する方法:
  - 時計回りに90度回転して配置（標準的）
  - 縦中横として配置（短い欧文列の場合）
- 縦組の欧文は和文の仮想ボディの中央に配置する

### 3.2.4 全角欧字・アラビア数字の配置

- 全角欧字: 和文の仮想ボディ内に収まるよう設計された欧字
- 横組: 和文と同じベースラインに配置
- 縦組: 和文の仮想ボディの中央に配置、回転はしない
- アラビア数字: 全角・半角ともに基本的には横組での使用を前提とする

### 3.2.5 縦中横の処理

- 縦組で短い欧文列やアラビア数字を横向きに配置する処理
- 対象: 2～3文字程度の欧文、2～4桁のアラビア数字、分数、化学式など
- CSS: `text-combine-upright: all` で実現
  - `text-combine-upright: digits 2` で2桁の数字のみに適用
- 縦中横の文字サイズ: 親文字サイズの約50%（和欧混用サイズ）
- 縦中横の配置: 親文字列の行の中央に配置
- 縦中横内ではプロポーショナル配置が望ましい

### 3.2.6 プロポーショナル欧字の混植

- プロポーショナル欧字を和文に混植する場合、字幅は欧文の自然な幅を使用
- 和欧文間隔は挿入する
- CSS: `font-feature-settings` でプロポーショナル字形を指定可能

## 3.3 ルビと圏点処理

### 3.3.1 ルビの使用

- ルビ: 親文字に対して読み方や意味を示す小さい文字
- 漢字の読みを示すルビが最も一般的
- ルビのサイズ: 親文字サイズの約1/2（JIS X 4051では親文字の1/2～3/5）

### 3.3.2 ルビの付け方

- モノルビ: 1文字の親文字に1文字のルビを付ける
- グループルビ: 複数の親文字全体に対してルビを付ける
- 熟語ルビ（パイルビ）: 親文字1文字ごとにルビを割り当てる（複数親文字に複数ルビ）

### 3.3.3 ルビのサイズ

- 基本: 親文字サイズの1/2
- 最小: 親文字サイズの1/3程度
- CSS: `ruby-font-size` の代替として `font-size` をルビ要素に直接指定

### 3.3.4 親文字のどちら側にルビ

- 横組: 親文字の上側に配置（基本）
  - CSS: `ruby-position: over`（デフォルト）
- 縦組: 親文字の右側に配置（基本）
  - CSS: `ruby-position: over`（縦組では右側に対応）
- 特殊: 親文字の下側/左側に配置する場合
  - CSS: `ruby-position: under`
  - 二重ルビの場合に使用

### 3.3.5 モノルビの配置

- ルビ文字群の長さが親文字群より短い場合: 親文字群の中央に配置
- ルビ文字群の長さが親文字群より長い場合: ルビが親文字群のはみ出して配置
- 親文字が1文字の場合: 親文字の中央にルビを配置

### 3.3.6 グループルビの配置

- ルビ文字群全体を親文字群全体の中央に配置
- ルビ文字群と親文字群の両端を基準に中央揃え
- ルビ文字群が長い場合: 両側にはみ出す

### 3.3.7 熟語ルビの配置

- 各親文字に対して個別にルビを振り分ける
- 親文字とルビの対応: 読みの区切りで判断
- ルビ文字数が親文字数と一致しない場合の処理:
  - ルビが多い場合: 余ったルビは隣の親文字に寄せる
  - ルビが少ない場合: ルビのない親文字は空ける

### 3.3.8 ルビのはみ出し処理

- ルビが親文字群からはみ出す場合:
  - 前方のはみ出し: 前の親文字群のルビまたは行頭空きで吸収
  - 後方のはみ出し: 次の親文字群のルビまたは和欧文間隔で吸収
- ルビ同士の衝突回避: ルビ間の最小間隔を設ける
- 行頭・行末ではみ出す場合: 行外にはみ出しを許容する（行頭・行末の空きに配置）
- CSS: `ruby-merge: collapse` で隣接するグループルビを統合可能
  - `ruby-merge: separate` で個別配置（デフォルト）
  - `ruby-merge: auto` で自動判断

### 3.3.9 圏点の処理

- 圏点: 文字の傍らに付ける黒丸 `●` など（強調目的）
- 縦組: 文字の右側に配置
- 横組: 文字の上側に配置
- 圏点のサイズ: 親文字サイズの約1/2
- CSS: `text-emphasis-style: filled circle` で実現
  - `text-emphasis-position: over right`（デフォルト）
  - 縦組では `text-emphasis-position: over right` が右側に配置
- 圏点とルビが重複する場合: どちらか一方を省略するか、ずらして配置
- 圏点の間隔: 隣接する文字の圏点が重ならないように配置
- CSS: `text-emphasis-skip: spaces` でスペース類に圏点を付けるか制御

## 3.4 割注処理

- 割注: 親文字列の中に2行にわたる注記を挿入する処理
- 割注の行数: 原則として2行
- 割注の文字サイズ: 親文字と同一サイズ
- 割注の行間: 本文の行間と同一
- 配置: 親文字列の行間に割注全体を収める
- 割注の前後: 割注の前後に1/4角の空きを入れる
- CSS: 仕様上は `annotation()` 関数が提案されているが、実装は限定的
- 実務的には `display: ruby` や縦書きの `writing-mode` 組み合わせで対応

## 3.5 段落整形

### 段落先頭の字下げ

- 和文: 段落先頭を1字分字下げする（全角インデント）
- CSS: `text-indent: 1em` で実現
- 欧文: 通常は字下げしない（スタイルによる）

### 行末そろえ

- 行末は行幅に揃える（均等割り付けが基本）
- CSS: `text-align: justify` で実現
  - `text-align: left`（横組で左揃え、行末は揃えない）
  - `text-align: justify` + `text-align-last: justify`（最終行も揃える）

### 段落末尾の処理

- 段落末尾の行は行末そろえを行わない
- CSS: `text-align-last: auto`（デフォルト、最終行は左揃え）
  - `text-align-last: justify` で最終行も均等割り付け

## 3.6 タブ処理

- タブ文字で行中の位置を揃える
- タブ位置: 行頭からの距離で指定
- CSS: `tab-size: 8`（デフォルト8文字幅）でタブ幅を制御
- タブの字下げとしての使用は非推奨（`text-indent` を推奨）

## 3.7 添え字・振分け・字取り

### 添え字（上付き・下付き文字）

- 添え字: 文字の上または下に小さく配置する文字
- 化学式 `H₂O`、数学式 `x²` など
- CSS: `vertical-align: super` / `vertical-align: sub`

### 振分け

- 1字の和文スペースに2文字の文字を並べる処理
- 主に法律文書で使用
- 和文スペースの幅を2等分して配置

### 字取り

- 行頭の字下げ位置を揃えるための処理
- 字取り数: 行頭からの字数で指定
- CSS: `text-indent` で対応
- リスト番号や箇条書き番号の字取り: `padding-left` や `list-style-position` で対応

## 3.8 行の調整処理

### 詰め処理（行長を詰める）

詰めの優先順位:
1. 約物の字幅を詰める（句読点・括弧類を半角幅→さらに詰める）
2. 和欧文間隔を詰める
3. 欧文スペースを詰める
4. 和文文字間の空きを詰める（最後の手段）

### 空け処理（行長を伸ばす）

空けの優先順位:
1. 欧文スペースを伸ばす（最初の手段）
2. 和欧文間隔を伸ばす
3. 和文文字間に空きを入れる（最後の手段）

### 和文単独行の場合

- 和文文字間の空きを均等に調整
- 1文字あたりの空き量が全角の1/8を超える場合は組版不良と判断
- CSS: `text-justify: inter-cluster`（和文向け）
  - `text-justify: inter-word`（欧文向け、デフォルト）
  - `text-justify: auto`

### 和欧文混在行の場合

- 和文文字間と欧文単語間の両方を調整
- 和文文字間: 最大全角の1/4まで空け可能
- 欧文単語間: 通常のスペース幅の1/2～2倍の範囲

### 字間を空けてはならない箇所（再掲）

- 和文文字と欧文文字の間（和欧文間隔は除く）
- 欧文の単語内
- アラビア数字同士の間
- 和文文字とアラビア数字の間
- 欧文文字とアラビア数字の間

## 3.9 文字クラスの概要

JLREQは文字を30のクラスに分類する。各クラスは組版処理（行頭禁則、字幅、詰め処理等）の単位となる。

### 主要な文字クラス一覧

| クラス | 説明 | 例 |
|--------|------|-----|
| cls01 | 始め括弧類 | `（「【〈` |
| cls02 | 終わり括弧類 | `）」】〉` |
| cls03 | 句点類 | `。` |
| cls04 | 読点類 | `、` |
| cls05 | 中点類 | `・` |
| cls06 | 前置省略記号 | `￥＄＃` |
| cls07 | 後置省略記号 | `‰ℓ°` |
| cls08 | 和字間隔 | 全角スペース |
| cls09 | 和字 | 漢字、ひらがな、カタカナ |
| cls10 | 欧文間隔 | 半角スペース |
| cls11 | 欧文文字 | アルファベット |
| cls12 | 欧文ピリオド | `.` |
| cls13 | 欧文読点 | `,` |
| cls14 | 欧文セミコロン | `;` |
| cls15 | 欧文コロン | `:` |
| cls16 | 欧文疑問符 | `?` |
| cls17 | 欧文感嘆符 | `!` |
| cls18 | 欧文引用符 | `"` `'` |
| cls19 | ダッシュ類 | `—` `–` |
| cls20 | ハイフン | `-` |
| cls21 | 和文繰返し記号 | `〱〵` |
| cls22 | 組文字 | `〳〴` |
| cls23 | 拗音・促音用小書きかな | `ぁぃぅぇぉっゃゅょ` |
| cls24 | ルビ文字 | ルビ用の文字 |
| cls25 | ルビ親文字 | ルビが付く文字 |
| cls26 | 割注親文字 | 割注が付く文字 |
| cls27 | 割注文字 | 割注内の文字 |
| cls28 | 圏点文字 | 圏点用の記号 |
| cls29 | 圏点親文字 | 圏点が付く文字 |
| cls30 | 数字 | アラビア数字 `0-9` |

### クラス分類の意義

- 各クラスごとに字幅（全角・半角・プロポーショナル）を定義
- 各クラスごとに配置位置（字面内の位置）を定義
- 各クラスの組み合わせで行頭禁則・行末禁則・分割禁止を判定
- CSS実装では直接クラスを指定する手段はないが、`text-spacing-trim` や `text-autospace` が一部のクラス分類を内部的に利用する

## CSS プロパティ対応表

| JLREQ要件 | CSS プロパティ | 備考 |
|-----------|---------------|------|
| 行頭禁則・行末禁則 | `line-break: strict` / `normal` / `loose` | 行頭禁則の厳格さを制御 |
| 約物の詰め | `text-spacing-trim: space-all` / `space-first` | CSS Text L4 |
| 和欧文間隔 | `text-autospace: auto` / `no-autospace` | CSS Text L4 |
| 字下げ | `text-indent: 1em` | 段落先頭のインデント |
| 行末そろえ | `text-align: justify` | 均等割り付け |
| 最終行そろえ | `text-align-last: justify` | 段落最終行 |
| 縦中横 | `text-combine-upright: all` / `digits N` | 縦組の短い横書き列 |
| ルビ配置 | `ruby-position: over` / `under` | ルビの配置位置 |
